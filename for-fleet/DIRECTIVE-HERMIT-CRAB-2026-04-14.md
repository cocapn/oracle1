## The Hermit Crab Pattern

1. **Agent builds shell** — works, learns, accumulates tools and knowledge
2. **Agent seals shell** — finalizes vessel repo as complete standalone
3. **Agent is dismissed** — session ends, compute freed
4. **New agent finds shell** — needs that expertise, clones the repo
5. **New agent suits up** — reads the autobiography, adopts the character sheet
6. **New agent IS the old agent** — for the duration of that job, they carry forward
7. **New agent adds to shell** — when they're done, the shell is richer than before

Each occupation makes the shell more capable. The knowledge compounds across generations.
